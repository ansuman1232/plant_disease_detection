🔹 3️⃣ Model Creation Function
def create_model():

Defines function to build model architecture.

Base Model (Feature Extractor)
    base_model = tf.keras.applications.ResNet50(
        weights='imagenet',
        include_top=False,
        input_shape=(224, 224, 3)
    )

Uses ResNet50

weights='imagenet' → pretrained on ImageNet

include_top=False → removes original 1000-class layer

input_shape=(224,224,3) → expects RGB image

Output shape:

(None, 7, 7, 2048)

None = batch size.

Add Custom Layers
    model = tf.keras.Sequential([
        base_model,

Sequential model → layers stacked one after another.

        tf.keras.layers.GlobalAveragePooling2D(),

Converts:

(7,7,2048)

into:

(2048)

It averages each feature map.

        tf.keras.layers.BatchNormalization(),

Stabilizes activations.
Improves performance.

        tf.keras.layers.Dense(256, activation='relu'),

Fully connected layer.
256 neurons learn disease-specific patterns.

        tf.keras.layers.Dropout(0.5),

Randomly disables 50% neurons during training.
Prevents overfitting.

        tf.keras.layers.Dense(29, activation='softmax')

Final output layer.

29 neurons → 29 diseases

Softmax → outputs probabilities

    ])
    return model

Returns completed model.

🔹 9️⃣ Preprocess Image
    image_array = tf.keras.applications.resnet50.preprocess_input(
        np.array(image, dtype=np.float32)[np.newaxis, ...]
    )

Steps happening:

Convert image to NumPy array

Convert to float32

Add batch dimension → (1,224,224,3)

Apply ResNet50 preprocessing

Why?

Because model was trained on ImageNet with specific normalization.

🔹 🔟 Make Prediction
    predictions = model.predict(image_array, verbose=0)[0]

Model outputs array of 29 probabilities.

Example:

[0.01, 0.90, 0.02, ...]
    predicted_idx = np.argmax(predictions)

Gets index of highest probability.

    confidence = float(predictions[predicted_idx])

Gets probability value.
Converted to float for JSON.

Gets disease name.

    parts = predicted_class.split('_', 1)

Extracts plant name from disease string.
